Tutorial: React Native para iPhone

Opção 2 - Aplicativo Nativo iOS via React Native

🎯 Objetivo

Desenvolver versão nativa do Simulador de Empréstimos para iPhone usando React Native, permitindo distribuição via App Store.

💰 Custos Envolvidos

  • Apple Developer Program: $99 USD/ano (obrigatório)
  • Mac/MacBook: Necessário para desenvolvimento iOS
  • Tempo de desenvolvimento: 2-4 semanas
  • App Store Review: 1-7 dias para aprovação

📋 Pré-requisitos Obrigatórios

  • Mac/MacBook com macOS 10.15+
  • Xcode 12+ instalado
  • Node.js 16+ e npm/yarn
  • React Native CLI
  • CocoaPods (gerenciador de dependências iOS)
  • Conta Apple Developer ativa
  • iPhone físico para testes (recomendado)

🔧 FASE 1: Configuração do Ambiente

Passo 1: Instalar Dependências

# Instalar Homebrew (se não tiver)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Instalar Node.js
brew install node

# Instalar Watchman (monitoramento de arquivos)
brew install watchman

# Instalar CocoaPods
sudo gem install cocoapods

Passo 2: Instalar Xcode

1. Abra a App Store no Mac

2. Procure por "Xcode"

3. Clique em "Instalar" (download ~10GB)

4. Após instalação, abra o Xcode

5. Aceite os termos e instale componentes adicionais

Passo 3: Configurar React Native

# Instalar React Native CLI globalmente
npm install -g @react-native-community/cli

# Verificar instalação
react-native --version

📱 FASE 2: Criação do Projeto

Passo 1: Inicializar Projeto React Native

# Criar novo projeto
npx react-native init MEEmprestimosApp

# Navegar para o diretório
cd MEEmprestimosApp

Passo 2: Instalar Dependências Necessárias

# Navegação
npm install @react-navigation/native @react-navigation/stack
npm install react-native-screens react-native-safe-area-context

# Componentes UI
npm install react-native-vector-icons
npm install react-native-paper

# Geração de PDF
npm install react-native-html-to-pdf

# Storage local
npm install @react-native-async-storage/async-storage

# Instalar dependências iOS
cd ios && pod install && cd ..

Passo 3: Estrutura Base do Projeto

Criar estrutura de pastas:

src/
├── components/ # Componentes reutilizáveis
├── screens/ # Telas da aplicação
├── utils/ # Funções auxiliares
├── styles/ # Estilos globais
└── navigation/ # Configuração de navegação

⚡ FASE 3: Desenvolvimento

Passo 1: Migrar Lógica de Negócio

Adaptar código JavaScript existente:

// src/utils/SimuladorCalculos.js
export class SimuladorCalculos {
calcularParcela(valor, juros, nParcelas, diasExtra = 0) {
// Migrar lógica existente do script.js
const jurosMensal = juros / 100;
const parcelaBase = valor * Math.pow(1 + jurosMensal, nParcelas) / nParcelas;
return parcelaBase;
}
}

// src/utils/ValidacaoRegras.js
export const validarLimites = (parcelas, juros) => {
// Migrar validações existentes
if (parcelas >= 1 && parcelas <= 3) {
return juros >= 15 && juros <= 30;
}
// ... resto das validações
};

🧪 FASE 4: Testes e Depuração

Passo 1: Executar no Simulador

# Abrir simulador iOS
npx react-native run-ios

# Ou especificar dispositivo
npx react-native run-ios --simulator="iPhone 14"

Passo 2: Testar em Dispositivo Real

1. Conecte iPhone via USB ao Mac

2. Configure certificados de desenvolvedor no Xcode

3. Execute:

npx react-native run-ios --device

Passo 3: Debug e Otimização

# Ativar debug no dispositivo
# Shake o dispositivo e selecione "Debug"

# React Native Debugger
npm install -g react-native-debugger

# Flipper (ferramenta de debug)
npx react-native doctor

🚀 FASE 5: Preparação para App Store

Passo 1: Configurar Apple Developer

1. Acesse developer.apple.com

2. Faça login com Apple ID

3. Inscreva-se no Developer Program ($99/ano)

4. Crie App ID para o projeto

5. Configure certificados de distribuição

Passo 2: Configurar App no Xcode

1. Abra ios/MEEmprestimosApp.xcworkspace no Xcode

2. Configure Bundle Identifier único

3. Selecione Team de desenvolvimento

4. Configure ícones e splash screen

5. Ajuste versão e build number

Passo 3: Build de Produção

# Gerar bundle de produção
npx react-native bundle --platform ios --dev false --entry-file index.js --bundle-output ios/main.jsbundle

# No Xcode, mudar schema para "Release"
# Product → Scheme → Edit Scheme → Run → Release

# Fazer archive
# Product → Archive

Passo 4: Submeter para App Store

1. No Xcode Organizer, selecione o archive

2. Clique em "Distribute App"

3. Selecione "App Store Connect"

4. Configure opções de distribuição

5. Upload para App Store Connect

6. Preencha metadados no App Store Connect

7. Submeta para review

⚠️ Desafios Comuns

Erros de Build:

  • Limpar cache: npx react-native start --reset-cache
  • Reinstalar pods: cd ios && rm -rf Pods && pod install
  • Limpar Xcode: Product → Clean Build Folder

Problemas de Certificado:

  • Verificar validade do certificado de desenvolvedor
  • Recriar provisioning profiles
  • Sincronizar certificados no Xcode

Rejeição da App Store:

  • Seguir Human Interface Guidelines
  • Implementar política de privacidade
  • Testar em dispositivos reais
  • Documentar funcionalidades para review
