===============================================================================
                            VISÃO GERAL
===============================================================================

O projeto ME EMPREENDIMENTOS evoluiu de um simples calculador de empréstimos 
para uma suite completa de ferramentas financeiras com mais de 130 iterações 
documentadas.

TECNOLOGIAS PRINCIPAIS:
• HTML5, CSS3, JavaScript (ES6+)
• Material Design 3 com temas claro/escuro
• Node.js para backend
• Socket.IO para chat em tempo real
• PostgreSQL (Neon) para banco de dados
• Progressive Web App (PWA)

MÓDULOS IMPLEMENTADOS:
1. Simulador de Empréstimos - Cálculos avançados com múltiplos sistemas de juros
2. Sistema de Chat com IA - OpenAI, Claude, Gemini, Grok
3. Análise de Documentos Financeiros - Upload e processamento automático
4. Hub Central - Navegação unificada entre módulos

===============================================================================
                        CONFIGURAÇÕES DO SISTEMA
===============================================================================

LIMITES DE TAXA DE JUROS:
• Parcelas 1-3: 15,00% - 30,00%
• Parcelas 4-15: 15,00% - 24,00%

SISTEMAS DE JUROS DISPONÍVEIS:
• Juros Simples - Cálculo linear tradicional
• Juros Compostos Diários - Cálculo exponencial por dia
• Juros Compostos Mensais - Cálculo exponencial por mês (padrão)
• Pro-rata Real - Cálculo individualizado por parcela

MENSAGENS DE ERRO (Português, Maiúsculas):
• Parcelas < 1: "NÚMERO DE PARCELAS INFERIOR AO MÍNIMO PERMITIDO."
• Parcelas > 15: "VOCÊ NÃO TEM PERMISSÃO PARA SIMULAÇÕES ACIMA DE 15 PARCELAS. 
  PARA SIMULAÇÕES SUPERIORES A 15 PARCELAS, CONSULTE MIGUEIS."
• Juros fora dos limites: "[N] PARCELA(S), A PORCENTAGEM MÍNIMA/MÁXIMA 
  PERMITIDA É DE X,XX %. PARA EMPRÉSTIMOS COM JUROS FORA DOS LIMITES 
  ESPECIFICADOS, CONSULTE MIGUEIS."
